快速生成Flink项目

  • 开发工具：idea+maven+git

步骤：

  1. 官网：https://ci.apache.org/projects/flink/flink-docs-release-1.8/dev/projectsetup/java_api_quickstart.html
  2. 本地创建目录：~/projects/flinkstudy

  3. 进入目录执行：curl https://flink.apache.org/q/quickstart.sh | bash -s 1.8.0；可以查看https://flink.apache.org/q/quickstart.sh返回的代码，参考该代码后面可部署新的项目环境

 PACKAGE=quickstart

 mvn archetype:generate                                \

   -DarchetypeArtifactId=flink-quickstart-java        \
   -DarchetypeVersion=${1:-1.8.0}                            \

 echo -e "\\n\\n"

  4. idea打开目录：创建batch和stream包，这个是学习用的，将批处理跟流处理的代码分开

  1. 从github官网搜索flink，然后看flink-example。这里找了java版的批处理和流处理的WordCount的例子。可直接执行。

小结：

  • github学习新东西，可以从example和单元测试的代码入手，因为会包含核心功能，且相对容易理解

  • Flink批处理和流处理的流程

    1. 命令行参数获取（可不带参数）

       final ParameterTool params = ParameterTool.fromArgs(args);

    2. 设置运行环境，是单机的jvm，还是单独的集群，还是on yarn等

       final ExecutionEnvironment env = ExecutionEnvironment.getExecutionEnvironment();

    3. 获取参数

       env.getConfig().setGlobalJobParameters(params);

    4. 获取输入：利用env封装的函数获取输入数据

    5. 处理输入：注意Map和flatMap的区别，Map是1个输入对应1个输出；flatMap是1个输入对应多个输出
    6. 输出：可根据命令行输出到某目录下，也可以直接控制台输出等
